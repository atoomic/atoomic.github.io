# Proof assets

Drop evidence here so it can be surfaced on the portfolio as proof of the "hire this person" claim
(see PRODUCT.md → **Conversion & proof**).

Requested:

- **Testimonials / recommendations** — text files (`.txt` / `.md`) or screenshots (`.png` / `.jpg`).
  Pull-quotes from colleagues, managers, or clients (e.g. surfaced from LinkedIn recommendations).
- **Quantified impact metrics** — team size led, module download counts, performance gains,
  adoption numbers, etc. A short `metrics.md` with one figure per line is ideal.

Reference files in this folder by relative path when wiring them into the site.
